Project

Library Engine

  • Design relational schema and create a database host application that interfaces with a backend SQL database implementing a Library Management System. Users of the system are understood to be librarians.
  • The application have the function to search books and their availability given any combination of ISBN, title, and authors, substring matching is suported.
  • In this app, librarians could manage account of all borrowers in the library system.
  • The application could help librarians to check out and check in books and system will calculate and pay fine of borrowers if the fail to return books on time.

Simulation Chat Room

  • Apply JAVA to simulate a char room system, though a server class, users could check other user's status like known, online, offline and send message to other users with different status.
  • Communication between processes is fulfilled by sockects. Samaphores are ultilized to enforce mutual exclusion between critical resource.

Experiments with a deep neural network using TensorFlow

  • Using part data of mnist data set as training data, construct diffrent networks to make learning accuracy as high as possible.
  • Gradient descent and AdamOptimizer combined with droupouts, regularization are experimented. Also convolutional layere is implemented in this project.
